# parcel-react

A minimal React boilerplate with [parcel as bundler](https://parceljs.org/)

## Installation

```git
git clone https://github.com/kingRayhan/parcel-react.git
cd parcel-react
npm install
npm start
```

**Build project**

```git
npm run build
```
